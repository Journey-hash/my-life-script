# 📱 如何在手机或其他电脑上访问

## 🚀 快速方案：使用 localtunnel（推荐临时使用）

### 步骤：

1. **确保开发服务器正在运行**
   ```bash
   # 如果还没运行，先启动
   npm run dev
   ```

2. **打开新的终端窗口，运行：**
   ```bash
   cd "/Users/lvcheng/Desktop/我的人生剧本-(my-life-script)"
   ./start-public.sh
   ```

3. **等待显示公网链接**
   会显示类似这样的链接：
   ```
   your url is: https://xxxx-xxxx-xxxx.loca.lt
   ```

4. **在手机或其他电脑的浏览器中打开这个链接**
   - 可以在任何网络访问（包括手机移动网络）
   - 不需要和电脑在同一 WiFi

### ⚠️ 注意事项：
- 每次运行可能会生成不同的链接
- 适合临时测试使用
- 链接在 localtunnel 进程关闭后会失效

---

## 🌐 稳定方案：使用 Cloudflare Tunnel（推荐长期使用）

### 步骤：

1. **安装 cloudflared（如果还没安装）**
   ```bash
   brew install cloudflare/cloudflare/cloudflared
   ```

2. **运行脚本**
   ```bash
   ./start-cloudflare.sh
   ```

3. **会显示一个 Cloudflare 的链接**
   例如：`https://xxxx-xxxx-xxxx.trycloudflare.com`

4. **在手机或其他电脑上打开这个链接**

### ✅ 优点：
- 链接更稳定
- 免费使用
- 适合长期使用

---

## 🎯 永久方案：部署到 Vercel（推荐生产环境）

如果你想要一个永久的、稳定的公网链接，建议部署到 Vercel：

1. **按照 `部署步骤.md` 中的说明操作**
2. **部署后会得到一个永久的 URL**
   例如：`https://my-life-script.vercel.app`
3. **这个链接可以永久使用，任何地方都能访问**

---

## 💡 推荐使用场景

- **临时测试**：使用 localtunnel（`./start-public.sh`）
- **短期使用**：使用 Cloudflare Tunnel（`./start-cloudflare.sh`）
- **长期使用**：部署到 Vercel（参考 `部署步骤.md`）
