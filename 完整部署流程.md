# 🚀 Vercel 完整部署流程（手动版）

## 第一步：注册/登录 Vercel

### 1. 打开浏览器，访问：
```
https://vercel.com/signup
```

### 2. 选择注册方式（推荐使用 GitHub）
- 点击 "Continue with GitHub"（推荐）
- 或使用邮箱注册

### 3. 完成注册/登录
- 如果使用 GitHub，会跳转到 GitHub 授权
- 授权后自动完成注册

---

## 第二步：在终端中登录 Vercel

### 1. 在终端运行：
```bash
cd "/Users/lvcheng/Desktop/我的人生剧本-(my-life-script)"
npx vercel login
```

### 2. 会显示一个链接，例如：
```
> Visit the following URL in your browser:
> https://vercel.com/cli/login?token=xxxxx
```

### 3. 复制这个链接，在浏览器中打开

### 4. 在浏览器中点击 "Authorize" 或 "Continue"

### 5. 回到终端，会显示：
```
Success! Logged in as [你的邮箱]
```

---

## 第三步：部署项目

### 1. 确保项目已构建（如果还没构建）
```bash
npm run build
```

### 2. 运行部署命令
```bash
npx vercel --prod
```

### 3. 按照提示回答（通常直接回车即可）：
```
? Set up and deploy? [Y/n] 
→ 直接回车（Y）

? Which scope? 
→ 直接回车（选择你的账号）

? Link to existing project? [y/N] 
→ 输入 N（首次部署）

? What's your project's name? 
→ 直接回车（使用默认名称）或输入自定义名称，例如：my-life-script

? In which directory is your code located? 
→ 直接回车（使用 ./）
```

### 4. 等待部署完成（通常 1-2 分钟）

### 5. 部署完成后会显示：
```
✅ Production: https://your-project-name.vercel.app
```

**这就是你的永久链接！** 🎉

---

## 第四步：配置环境变量（重要！）

### 1. 访问 Vercel Dashboard
打开浏览器，访问：
```
https://vercel.com/dashboard
```

### 2. 找到你的项目
- 在项目列表中点击你的项目名称

### 3. 进入设置
- 点击顶部的 **Settings** 标签
- 在左侧菜单中找到 **Environment Variables**

### 4. 添加环境变量
- 点击 **Add New** 按钮
- 填写：
  - **Key**: `GEMINI_API_KEY`
  - **Value**: 你的 Gemini API Key（从 `.env.local` 文件中复制）
  - **Environment**: 勾选所有选项（Production、Preview、Development）
- 点击 **Save**

### 5. 重新部署
配置环境变量后，需要重新部署：

**方法 1：通过 Dashboard（推荐）**
- 在项目页面，点击顶部的 **Deployments** 标签
- 找到最新的部署
- 点击右侧的 **⋯**（三个点）
- 选择 **Redeploy**
- 确认重新部署

**方法 2：通过命令行**
```bash
npx vercel --prod
```

---

## 第五步：完成！

### 你的永久链接
部署完成后，你会得到一个永久 URL，例如：
```
https://my-life-script.vercel.app
```

### 这个链接的特点：
- ✅ **永久有效** - 不会过期
- ✅ **任何网络可访问** - 手机、其他电脑都能访问
- ✅ **自动 HTTPS** - 安全连接
- ✅ **全球 CDN** - 访问速度快

### 测试访问
在手机或其他电脑的浏览器中打开这个链接，应该能看到你的 H5 页面！

## ❓ 常见问题

### Q: 部署后显示 404 错误？
A: 检查 `vercel.json` 文件是否存在，确保有正确的路由配置

### Q: AI 功能不工作？
A: 检查是否配置了环境变量 `GEMINI_API_KEY`，并重新部署

### Q: 如何修改项目名称？
A: 在 Vercel Dashboard → Settings → General → Project Name

### Q: 如何查看部署日志？
A: 在 Vercel Dashboard → Deployments → 点击部署 → 查看 Logs
