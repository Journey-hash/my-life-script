## 🚀 快速部署（3 步完成）

### 步骤 1：运行部署脚本
在终端运行：
```bash
cd "/Users/lvcheng/Desktop/我的人生剧本-(my-life-script)"
./deploy-to-vercel.sh
```

### 步骤 2：登录 Vercel（首次需要）
- 脚本会自动打开浏览器
- 如果没有账号，点击 "Sign Up" 注册（免费）
- 如果有账号，直接登录
- 登录成功后，终端会显示 "Success! Logged in as [你的邮箱]"

### 步骤 3：按照提示完成部署
- **Set up and deploy?** → 直接回车（Y）
- **Which scope?** → 直接回车（选择你的账号）
- **Link to existing project?** → 输入 `N`（首次部署）
- **What's your project's name?** → 直接回车（使用默认名称）或输入自定义名称
- **In which directory is your code located?** → 直接回车（使用 `./`）

部署完成后会显示你的永久 URL！

## ⚙️ 配置环境变量（重要！）

部署完成后，**必须**配置 API Key，否则 AI 功能无法使用：

### 方法 1：通过 Vercel Dashboard（推荐）

1. 访问 https://vercel.com/dashboard
2. 找到你的项目（项目名称）
3. 点击项目进入详情页
4. 点击 **Settings** → **Environment Variables**
5. 点击 **Add New**
6. 填写：
   - **Key**: `GEMINI_API_KEY`
   - **Value**: 你的 Gemini API Key（从 `.env.local` 文件中复制）
   - **Environment**: 勾选 `Production`、`Preview`、`Development`（建议全选）
7. 点击 **Save**

### 方法 2：通过命令行

```bash
npx vercel env add GEMINI_API_KEY production
# 然后粘贴你的 API Key
```

### 重新部署

配置环境变量后，需要重新部署：

**方法 1：通过 Dashboard**
- 进入项目 → **Deployments**
- 点击最新部署右侧的 **⋯** → **Redeploy**

**方法 2：通过命令行**
```bash
npx vercel --prod
```

## 💡 常见问题

### Q: 部署后显示错误？
A: 检查是否配置了环境变量 `GEMINI_API_KEY`

### Q: 如何修改项目名称？
A: 在 Vercel Dashboard → Settings → General → Project Name

### Q: 如何删除部署？
A: 在 Vercel Dashboard → Settings → Delete Project

### Q: 免费版有限制吗？
A: 个人项目完全免费，有足够的带宽和构建次数
