# 🇨🇳 中国大陆访问部署方案

## 问题分析

Vercel 在中国大陆访问可能不稳定或被限制，导致手机无法打开。以下是几个解决方案：

---

## 方案 1：使用 Vercel + 国内 CDN（推荐）

### 使用 Cloudflare 加速（如果 Vercel 链接是 `*.vercel.app`）

1. 在 Vercel Dashboard 中，进入项目 → Settings → Domains
2. 添加自定义域名（如果有的话）
3. 使用 Cloudflare 作为 DNS，可以改善访问速度

### 使用国内 CDN 服务

可以考虑使用：
- **又拍云 CDN**
- **七牛云 CDN**
- **阿里云 CDN**

---

## 方案 2：部署到国内平台（最稳定）

### 选项 A：部署到阿里云 OSS + CDN

1. **创建阿里云账号**（如果有）
2. **开通 OSS 服务**
3. **上传 dist 目录到 OSS**
4. **配置 CDN 加速**
5. **绑定自定义域名**

### 选项 B：部署到腾讯云 COS + CDN

1. **创建腾讯云账号**
2. **开通 COS 对象存储**
3. **上传 dist 目录**
4. **配置 CDN**
5. **绑定域名**

### 选项 C：部署到华为云 OBS

类似阿里云和腾讯云的流程。

---

## 方案 3：使用 GitHub Pages + 国内镜像（免费）

### 步骤：

1. **将项目推送到 GitHub**
2. **使用 GitHub Actions 自动部署**
3. **使用国内 GitHub 镜像或 CDN 加速**

---

## 方案 4：使用 Netlify（可能比 Vercel 好一些）

Netlify 在中国大陆的访问可能比 Vercel 稍好：

```bash
# 安装 Netlify CLI
npm install -g netlify-cli

# 登录
netlify login

# 部署
netlify deploy --prod --dir=dist
```

---

## 方案 5：自建服务器（最灵活）

### 使用国内云服务器：

1. **购买云服务器**（阿里云、腾讯云、华为云等）
2. **安装 Nginx**
3. **上传 dist 目录**
4. **配置 Nginx 静态文件服务**
5. **绑定域名和备案**（如果需要）

---

## 🚀 快速解决方案（推荐）

### 如果只是临时测试，可以尝试：

1. **检查手机网络**
   - 尝试切换 WiFi 和移动数据
   - 尝试使用 VPN（如果合法使用）

2. **使用 Vercel 的自定义域名功能**
   - 如果有自己的域名，可以绑定到 Vercel
   - 使用国内 DNS 解析

3. **使用 Cloudflare Pages**（免费）
   - 可能比 Vercel 在中国访问更稳定

---

## 💡 最佳实践建议

对于需要在中国大陆稳定访问的项目，建议：

1. **短期方案**：使用国内云服务商的静态网站托管
   - 阿里云 OSS + CDN
   - 腾讯云 COS + CDN

2. **长期方案**：购买国内服务器 + 域名备案
   - 完全可控
   - 访问速度最快
   - 需要备案

---

## 📝 具体操作步骤

### 如果选择阿里云 OSS：

1. 登录阿里云控制台
2. 开通 OSS 服务
3. 创建 Bucket，设置静态网站托管
4. 上传 dist 目录中的所有文件
5. 配置 CDN 加速（可选）
6. 获取访问地址

### 如果选择腾讯云 COS：

1. 登录腾讯云控制台
2. 开通 COS 服务
3. 创建存储桶，开启静态网站
4. 上传 dist 目录
5. 配置 CDN（可选）
