## 📋 接下来需要你手动完成的步骤：

### 1. 登录 Vercel
在终端运行：
```bash
cd "/Users/lvcheng/Desktop/我的人生剧本-(my-life-script)"
npx vercel login
```

这会：
- 打开浏览器
- 提示你登录 Vercel（如果没有账号，会提示注册，免费）
- 登录成功后，终端会显示 "Success! Logged in as [你的邮箱]"

### 2. 部署项目
登录成功后，运行：
```bash
npx vercel --prod
```

按照提示操作：
- **Set up and deploy?** → 输入 `Y` 或直接回车
- **Which scope?** → 选择你的账号（通常直接回车）
- **Link to existing project?** → 输入 `N`（首次部署）
- **What's your project's name?** → 直接回车使用默认名称，或输入自定义名称
- **In which directory is your code located?** → 直接回车（使用当前目录 `./`）

### 3. 配置环境变量（重要！）
部署完成后，需要配置 API Key：

1. 访问 [Vercel Dashboard](https://vercel.com/dashboard)
2. 找到你的项目（项目名称）
3. 进入 **Settings** → **Environment Variables**
4. 添加新变量：
   - **Key**: `GEMINI_API_KEY`
   - **Value**: 你的 Gemini API Key（从 `.env.local` 文件中复制）
   - **Environment**: 选择 `Production`、`Preview`、`Development`（建议全选）
5. 点击 **Save**

### 4. 重新部署
配置环境变量后，需要重新部署：
```bash
npx vercel --prod
```

或者：
- 在 Vercel Dashboard 中，进入项目 → **Deployments**
- 点击最新的部署右侧的 **⋯** → **Redeploy**

### 5. 完成！
部署完成后，你会得到一个 URL，例如：
- `https://my-life-script.vercel.app`

这个 URL 可以在任何网络（包括手机）访问！

---

## 💡 提示
- 首次部署可能需要 2-3 分钟
- 如果遇到问题，可以查看 Vercel Dashboard 中的部署日志
- 环境变量配置后，下次部署会自动使用
